# Animation packs

This document lists animation packs that include custom descriptions. It also notes whether these packs have been improved with better actions, tags, or other enhancements.

Patch - if it requires patches for scenes or some additional notes
Verion - which version of original pack got OStimNet updates
Custom Descriptions - whole pack was processed and its animations got custom descriptions
Improved Actions/Tags - scenes got improved tags/actions in their core mod or through patches
Updated intents/positions - pack's scenes are processed and marked with correct intents and revisited positions(for example before most packs used `missionary` position, OStimNet expands it to `missionary`, `anvil`, `butterfly`, `piledriver`, `matingpress`)

Special mods for undressing functionality:
- [OStim Aniamtions](https://www.nexusmods.com/skyrimspecialedition/mods/98163) mod for second actor undressing themself and then undressing first actor. Fast animations sequence.
- [NexusMods Open Animations Romance and Erotica*](https://www.nexusmods.com/skyrimspecialedition/mods/98732) for `OARE` undressing option - second actore undresses themself on their own, first actor helps to undress top, then second actor undresses first. Normal speed.
- [OStim OSA animations port](https://www.nexusmods.com/skyrimspecialedition/mods/158912) for `OSA` undressing options. First actor helps second actor to undress fully, then second actor undresses first. Slowest option.
